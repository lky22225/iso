# [Responsive Homepage UI Design]

## 1. Overall Concept
- Clean corporate SaaS style
- Education/consulting hybrid layout
- Conversion-focused structure with soft visual engagement
- Balanced composition of text + human imagery

---

## 2. Layout Structure
- Max width: 1200px centered
- 12-column responsive grid
- Section spacing: 80px desktop / 40px mobile
- Card-based modular layout

---

## 3. Color System (Strict Mode)

### Background (BG)
- BG Primary: #F5F7FA
- BG Secondary: #E9F1F7

### Text
- Text Primary: #1F2A37
- Text Secondary: #6B7280

### Accent
- Accent Primary: #4F46E5
- Accent Secondary: #7DD3FC

---

## 4. Typography
- Font: Sans-serif (Inter or similar)
- H1: 48px / bold
- H2: 32px / semibold
- Body: 16px / regular
- Line height: 1.6

---

## 5. UI Components

### Header
- Left: Logo
- Center: Navigation (Home, About, Services, Contact)
- Right: Primary CTA button (rounded, filled)

### Hero Section
- Left: Headline + subtext + CTA button
- Right: Human image (cutout with soft shadow)
- Background: gradient using BG Secondary + Accent Secondary
- Decorative floating shapes (rounded blobs)

### Feature Icons Row
- 4-column layout
- Rounded icon containers
- Soft pastel backgrounds

### Content Cards Section
- Horizontal card with image + text
- Rounded corners (16px)
- Subtle shadow

### Process Section
- Numbered steps (1–3 or 1–6)
- Vertical or grid layout
- Circle number indicators

### Course/Grid Section
- Image overlay cards
- Text appears on hover or gradient overlay
- 2x3 grid layout

### Testimonial Section
- Card carousel
- Profile image + quote
- Soft shadow + rounded corners

### CTA Banner
- Full-width rounded container
- Gradient background
- Centered text + button

### Footer
- 3–4 column layout
- Logo + links + contact + newsletter input

---

## 6. Shapes & Graphic Style
- Rounded corners (12–20px)
- Soft shadows (low opacity)
- Gradient overlays (subtle)
- Floating abstract shapes (circle, blob)
- Human-centered imagery

---

## 7. Interaction
- Hover: slight elevation + shadow increase
- Button hover: darker accent shade
- Smooth transitions (0.3s ease)

---

## 8. Tone & Manner
- Friendly and professional
- Trust-driven but approachable
- Educational and supportive

---

## 9. Responsive Rules
- Tablet: 2-column grid
- Mobile: single column
- Hero image stacks below text
- Reduce padding by 50% on mobile

---

## 10. Conversion Elements
- Primary CTA repeated every 2–3 sections
- Secondary CTA for low-commitment actions
- Clear visual hierarchy for decision flow

---

## 11. 기술 아키텍처
### 기술 스택
- **Frontend**: Next.js 14+ (App Router), React, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: JSON 파일 시스템
- **스타일링**: Tailwind CSS (반응형 디자인)
- **배포**: Vercel
