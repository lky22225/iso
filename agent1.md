# Agent1 조사 정리

## 목적
한국에서 ISO 인증서비스를 제공하는 회사들의 홈페이지를 벤치마킹하여, 디자인/메뉴체계/콘텐츠 구조를 도출한다.

## 조사 대상
- 한국품질재단(KFQ): https://www.kfq.or.kr/
- 한국경영인증원(KMR): https://www.ikmr.co.kr/
- ITS인증원: https://itscert.or.kr/
- 한국표준협회(KSA): https://ksa.or.kr/
- 한국ISO인증기술원: https://www.koreaiso.kr/
- 한국기업인증센터(KECC): http://iso-kecc.com/
- 국제표준경영원(ISMC): http://www.isocenter.co.kr/
- 한국글로벌인증원(KGC): https://www.kgcert.com/

## 회사별 핵심 포인트
### 1) KFQ
- 경영시스템 인증, 제품 인증, ESG, 교육 등 다각 서비스 구조.
- 메인에서 서비스 카드 + 소식/행사 노출.
- 문의/상담 CTA 중심.

### 2) KMR
- 인증/진단평가/교육/시상 등 확장형 포지셔닝.
- 실적 수치, 다운로드 자료(회사소개서/로고) 노출.
- 신뢰도 강조형 메인 구성.

### 3) ITS인증원
- 인증분야 소개 + 신청/조회/다운로드 허브 구조.
- 인증기업/심사원/교육생 조회 기능을 전면 배치.
- 실무 전환(신청) 중심 UX가 강함.

### 4) KSA
- 표준/인증/교육/지수·포상 등 포털형 사이트.
- ISO 단일 세일즈보다 기관 종합 정보 제공 성격.

### 5) 한국ISO인증기술원
- 규격별 상세 링크 다수, 상담 유도형 카피 강함.
- 사례/FAQ/리뷰 기반 리드 전환 구조.

### 6) KECC
- 정형 메뉴보다 “인증 추천” 퍼널 중심.
- 사용자 질문 기반 맞춤 인증 추천 흐름.

### 7) ISMC
- 회사소개/인증컨설팅/제품인증/교육/고객센터의 전형적 구조.
- 표준별 서비스 설명 + 문의 전환형.

### 8) KGC
- 인증원 소개, 인증절차, 신청문의, 이의제기, 로고다운로드 중심.
- 비교적 컴팩트한 정보 제공형 메인.

## 공통 인사이트
- 핵심 전환 흐름: `인증분야 소개 -> 절차 안내 -> 문의/신청`.
- CTA는 전화/문의/견적이 중심이며 상단 고정 노출 비중이 큼.
- 신뢰요소로 인증기업 조회, 심사원 정보, 실적 수치, 로고/문서 다운로드를 활용.
- 9001/14001/45001/27001을 기본축으로 제시하고 확장 규격을 병렬 노출.

## 차별화 패턴
- 진단형 퍼널(맞춤 추천형)
- 데이터 신뢰형(실적 수치/이력 강조형)
- 운영툴형(신청/조회/다운로드 허브형)
- 포털형(인증 외 표준/교육 확장형)
- 상담형(문제 공감 + 전화 전환형)
